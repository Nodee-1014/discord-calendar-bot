# 🌟 マストワンシステム使い方ガイド

## 📌 マストワンシステムとは？

**今日やるべきタスクの中で、最も重要な「主役タスク」を1つだけ選んで☆マークをつけるシステムです。**

- 常に**1つだけ**☆をつけられる（Must One = マストワン）
- `/progress`で最優先表示
- 集中力を高め、本当に重要なタスクに注目

---

## 🚀 使い方

### 1️⃣ 主役タスクを設定する

```
/must_one [タスク名の一部]
```

**例:**
```
/must_one レポート作成
```

**結果:**
- カレンダーのタスク名が `☆ レポート作成 ★★★ 2h` のように変更される
- 既に他のタスクに☆がついていた場合、自動的に削除される

---

### 2️⃣ 進捗レポートで確認

```
/progress
```

**表示例:**
```

達成率: 63% ██████████░░░░░░░░░░

🌟 今日の主役タスク:
☆ レポート作成 ★★★ 2h 14:00-16:00

✅ 完了タスク (4個):
• 会議 ✓ 09:00-10:00

• データ分析 ★★ 1h 11:00-12:00
• 資料準備 ★ 30m 13:00-13:30
...
```

## 💡 活用シーン

### ✅ こんな時に使おう

1. **タスクが多すぎて優先順位が分からない**
   - ☆をつけて「今日はこれだけは絶対やる！」と決める

2. **複数の重要タスクがあるが、一番重要なものを明確にしたい**
   - A（★★★）のタスクが複数ある中で、最優先の1つを選ぶ

3. **朝イチで今日の目標を確認したい**
   - `/progress`で☆タスクが最上位に表示される

4. **チームで共有カレンダーを使っている**
   - 自分の主役タスクを明示して、周りに重要度を伝える

## 🎯 マストワンシステムのルール

### ✨ 特徴

| 項目 | 説明 |
|------|------|
| **☆の数** | 常に1つだけ（マストワン） |
| **設定方法** | `/must_one [タスク名]` |
| **自動削除** | 新しい☆を設定すると、古い☆は自動削除 |
| **完了タスク** | 完了マーク（✓）がついたタスクには☆をつけられない |
| **優先度との違い** | ★★★（A）は複数可、☆は1つだけ |

### ⚠️ 注意点

- **☆は未完了タスクにのみ設定可能**
  - 完了済み（✓付き）のタスクには設定できません

- **1つのタスクにしか☆をつけられない**
  - 新しい☆を設定すると、前の☆は自動的に消えます

- **手動でカレンダーに☆を追加してもOK**
  - Googleカレンダーで直接タイトルに☆を追加しても認識されます

## 🔄 ワークフロー例

### 朝のルーティン

```
1. /progress で今日のタスクを確認
2. 最も重要なタスクを1つ選ぶ
3. /must_one [タスク名] で主役タスクに設定
4. 主役タスクを最優先で実行！
```

### 途中で変更する場合

```
1. 状況が変わって別のタスクが最重要になった
2. /must_one [新しいタスク名] で☆を付け替え
3. 自動的に前の☆は消えて、新しいタスクに☆がつく
```

## 📊 既存機能との連携

### ⭐ 優先度システムとの併用

| マーク | 意味 | 設定方法 | 複数可 |
|--------|------|----------|--------|
| **★★★** | A優先度（高） | `/t2g タスク 1h A` | ✅ |
| **★★** | B優先度（中） | `/t2g タスク 1h B` | ✅ |
| **★** | C優先度（低） | `/t2g タスク 1h C` | ✅ |
| **☆** | 今日の主役（マストワン） | `/must_one タスク` | ❌（1つのみ） |

### 🎯 使い分け

```
★★★ = 重要度が高いタスク（複数あってもOK）
☆ = 今日の主役タスク（1つだけ）
```

**例:**
```
09:00-10:00 ☆ レポート作成 ★★★ 1h  ← 今日の主役かつA優先度
11:00-12:00 プレゼン準備 ★★★ 1h   ← A優先度だが主役ではない
13:00-14:00 メール返信 ★★ 1h      ← B優先度
```

## 🛠️ トラブルシューティング

### Q1: ☆がつかない

**原因:**
- タスク名が部分一致しない
- 完了済み（✓付き）タスクに設定しようとしている

**解決策:**
```
# タスク名の一部でOK
/must_one レポート  # 「レポート作成」にもマッチ

# 完了タスクには設定できない
/undone レポート    # 先に完了を解除
/must_one レポート  # その後☆を設定
```

### Q2: 複数のタスクに☆をつけたい

**回答:**
マストワンシステムは**1つだけ**に集中するための機能です。
複数の重要タスクには、優先度（A/B/C → ★★★/★★/★）を使ってください。
